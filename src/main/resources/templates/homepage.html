Informações do Projeto

Esta aplicação foi desenvolvida utilizando o framework Spring Boot integrado ao motor de templates Thymeleaf e persistência de dados via Spring Data JPA.

  • Arquitetura: Camadas bem definidas divididas em Model, Repository, Service e Controller.
  • Design Pattern: Implementação do padrão comportamental Strategy com uma estrutura de Factory para computar tetos de crédito proporcionais à renda informada.
  • Interface Reutilizável: Componentização de fragmentos HTML promovendo reuso visual completo.
